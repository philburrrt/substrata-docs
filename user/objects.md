# Objects

## Models

The following 3d Model file types can be uploaded and scripted in Substrata. 

- GLTF 
- OBJ
- GLB
- VOX
- STL
- IGMESH

Find free 3d models like homes, vehicles and gardening at sites like: 
- [Sketchfab](https://sketchfab.com/feed)
- [Booth](https://booth.pm/en)
